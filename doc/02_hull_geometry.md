# 設計書 02 — 船体ジオメトリ設計

対象ファイル: `js/yamato/hull.js`

---

## 1. 座標系

```
     +Y（上）
      |
      |
      +--------+X（艦首方向）
     /
    +Z（右舷方向）

原点: 艦中央（全長中点）の水線面
水線: y = 0
単位: 1 unit = 1 m
```

---

## 2. 基本定数

| 定数 | 値 | 意味 |
|------|----|------|
| `LOA` | 263 m | 全長（Length Over All） |
| `HALF` | 131.5 m | 中点オフセット |
| `BEAM_HALF` | 19.45 m | 最大半幅（全幅 38.9 m） |

艦尾が x = −131.5、艦首が x = +131.5。

---

## 3. 無次元座標 u

船体プロファイルはすべて `u ∈ [0, 1]` で定義される。

```
u = (x + HALF) / LOA
u = 0 : 艦尾端
u = 1 : 艦首端
```

変換関数 `uOf(x)` で任意の x 座標から u を得る。

---

## 4. 船型プロファイル関数

### 4.1 水線面半幅 `halfBeam(u)`

```
u > 0.62: 艦首の絞り込み   f = 1 − t^1.6  (t = (u−0.62)/0.38)
u < 0.42: 艦尾の絞り込み   f = sin(t·π/2)^0.8 (巡洋艦型船尾)
半幅 = BEAM_HALF × max(f, 0.015)
```

- 中央部（0.42 ≤ u ≤ 0.62）は最大幅 19.45 m
- 艦首は急峻に絞り込まれ、球状艦首（バルバスバウ）へ接続
- 艦尾はサインカーブで滑らかに絞り（三軸・四軸スクリュー配置に対応）

### 4.2 甲板高さ `deckY(u)` — シア曲線

```
基準高さ: 8.7 m
u > 0.7: 艦首シア   +6.8·((u−0.7)/0.3)²    （艦首に向けて反り上がる）
u < 0.15: 艦尾      −1.7·((0.15−u)/0.15)^1.5  （艦尾は低くなる）
```

### 4.3 キール深さ `keelY(u)`

```
基準深さ: −10.4 m
u > 0.8: 艦首立ち上がり  +9.4·((u−0.8)/0.2)^1.7
u < 0.18: 艦尾カウンター  +9.0·((0.18−u)/0.18)^1.8
```

### 4.4 断面形状パワー

断面の「膨らみ」を制御するパラメータ。

```javascript
sectionPower(u)  = 0.35 + 1.45 · |u − 0.5|²     // サイド（Z 方向）
bottomPower(u)   = 1.2  + 1.4  · (1 − |u − 0.5|^1.5)  // 底面（Y 方向）
```

- 中央部: sectionPower 小 → 箱型断面（大排水量確保）
- 艦首尾: sectionPower 大 → V 型断面（波切り性能）

## 5. 船体メッシュ生成アルゴリズム `buildHullMesh()`

### 5.1 グリッド定義

| パラメータ | 値 | 説明 |
|-----------|----|----|
| STATIONS | 170 | 長手方向分割数（ステーション） |
| SECTS | 26 | 半断面分割数 |
| RING | 53 | 1 ステーションあたりの頂点数（= SECTS×2+1） |

総頂点数 ≈ 171 × 53 = 9,063  
総三角形数 ≈ 170 × 52 × 2 = 17,680

### 5.2 断面頂点配置

各ステーション i について、リング内の頂点 j を以下で配置:

```
t = j / (RING − 1)          // 0 = 右舷甲板縁, 0.5 = キール, 1 = 左舷甲板縁
side = (t < 0.5) ? +1 : −1  // 右舷 / 左舷
s    = 1 − |t − 0.5| × 2   // 1 = 舷端, 0 = キール

Z = side × halfBeam(u) × sin(s·π/2)^sectionPower(u)
Y = keelY(u) + (deckY(u) − keelY(u)) × (1 − cos(s·π/2)^bottomPower(u))
```

### 5.3 頂点カラー（塗り分け）

頂点の Y 座標に基づき 3 段階で塗り分ける。

| Y 範囲 | 色 | 用途 |
|--------|----|------|
| y < −1.1 | `rgb(0.45, 0.13, 0.11)` 赤 | 艦底防汚塗料 |
| −1.1 ≤ y < 1.1 | `rgb(0.07, 0.07, 0.08)` 黒 | 喫水線帯 |
| y ≥ 1.1 | `rgb(0.56, 0.58, 0.61)` 灰 | 軍艦色（呉海軍工廠グレー） |

マテリアル `hull` は `vertexColors: true` で受け取る。

### 5.4 UV 設定

```
UV.x = u × 30   // 長手方向：30 タイル
UV.y = t × 4    // 断面周方向：4 タイル
```

鋼板テクスチャの継ぎ目が適切な間隔で繰り返される。

## 6. 甲板ストリップ `buildDeckStrip()`

任意の X 範囲に甲板面を生成する汎用関数。

| 引数 | 意味 |
|------|------|
| `x0, x1` | 長手方向範囲 |
| `inset` | 舷側からの控え量（デフォルト 0.35 m） |
| `yOff` | 浮かせ量（重複回避用） |
| `segs` | 長手方向分割数（デフォルト 120） |

各位置の甲板高さは `deckY(u)` から取得。幅方向（ACROSS=10）に頂点を展開し、**キャンバー**（中央の盛り上がり）を付与:

```
Y = deckY(u) − 0.06 + 0.3 × (1 − f²) + yOff   (f: −1〜+1)
```

## 8. 付属構造物

### 8.1 バルバスバウ（球状艦首）

```javascript
SphereGeometry(1, 20, 14)
scale: (4.2, 2.0, 1.7)
position: (127.5, −6.8, 0)
```

水中の波形抵抗を低減する球状突起。スケールで楕円体化。

### 8.2 波除け（ブレークウォーター）

前甲板の V 字型波除け板。port / starboard 各 1 枚。

```
基点 A = (83, 0, 0),  先端 B = (74.5, 0, ±5.6)
BoxGeometry(9.6, 1.15, 0.18)
角度: Math.atan2(dZ, dX) で自動計算
```

### 8.3 推進システム `buildPropulsion()`

#### スクリューシャフト（4 軸）

| 軸 | x | z | y |
|----|---|---|---|
| 内右 | −108 | +3.9 | −6.6 |
| 内左 | −108 | −3.9 | −6.6 |
| 外右 | −99 | +7.2 | −6.0 |
| 外左 | −99 | −7.2 | −6.0 |

各軸: シャフト管（tubeGeometry）+ 支持ブラケット

#### プロペラ（3 翼）

```javascript
hub: CylinderGeometry(0.42, 0.5, 1.2, 12)  // ハブ
blade: SphereGeometry(1, 12, 8)
  scale: (0.1, 1.45, 0.62)                  // 翼形に変形
  各翼: 120° 間隔、ピッチ角 0.5 rad
```

マテリアル `bronze`（マンガン青銅、metalness 1.0）

#### 舵

| 種別 | 寸法 | 位置 |
|------|------|------|
| 主舵 | 4.6 × 6.2 × 0.55 m | x=−119, y=−4.6 |
| 副舵 | 3.2 × 4.6 × 0.50 m | x=−110, y=−5.4 |

#### ビルジキール

船体ローリングを抑制する縦翼板。port / starboard 各 1 枚。

```
BoxGeometry(85, 0.3, 1.4)
position.z = ±17.2
rotation.x = ±0.7 rad
```
