# ⚓ 戦艦大和 3D インタラクティブビューア

Three.js で戦艦大和（IJN YAMATO）をフルスクラッチでモデリングした、ブラウザで動く 3D ビューアです。外部 3D モデルや画像アセットを一切使わず、船体からテクスチャまですべてコードで生成しています ✨

## ✨ 機能・特徴

- **プロシージャル生成の大和** — 船体ロフト・46cm 三連装主砲 ×3・副砲・塔型艦橋・煙突・高角砲・25mm 機銃・探照灯・カタパルト・クレーン・錨鎖・手すり・菊花紋章・21号/13号電探まで再現（約 11 万ポリゴン）
- **リアル描画** — PBR マテリアル＋Canvas 動的テクスチャ（鋼板の継ぎ目・リベット・木甲板の板目）、大気散乱の空、波シェーダの海、太陽光と影
- **砲塔・砲身アニメーション** — 主砲・副砲（旋回 ±120〜150°／俯仰 −5°〜+45°）に加え 12.7cm 高角砲 12 基（旋回 ±120°／俯仰 −5°〜+75°）を共通トグルで自動駆動。射界制限でめり込みを防止
- **精密な塔型艦橋** — 窓枠・手すり・ラッタル・22号電探ホーン・測距儀光学筒・張線まで再現（約 13.4 万ポリゴン）
- **煙突の煙** — 240 パーティクルの煙エフェクト（リアル描画時のみ）
- **ワイヤーフレーム描画** — ワンタッチで設計図風（青写真）表示に切替（ボタン or `W` キー）
- **自由なカメラ操作** — 回転・移動・拡大縮小（OrbitControls）＋自動回転モード
- **完全自己完結** — ビルド不要・アセットファイル不要。海面の法線マップも実行時に生成

## 🛠 使用技術

- HTML / CSS / JavaScript（ES Modules）
- [Three.js](https://threejs.org/) r160（CDN / importmap 読み込み）
  - `Sky`（大気散乱）・`Water`（海面）・`OrbitControls`
- Canvas 2D API（テクスチャのプロシージャル生成）

## 🚀 ローカルで動かす方法

ES Modules を使うため、ローカルサーバ経由で開きます。

```bash
# いずれか好きな方法で
npx http-server -p 8173 .
# または
python -m http.server 8173
```

ブラウザで `http://localhost:8173` を開くと大和が表示されます 🚢

### 操作方法

| 操作 | 動作 |
|------|------|
| 左ドラッグ | 回転 |
| 右ドラッグ | 移動 |
| ホイール | 拡大縮小 |
| `W` キー / ボタン | リアル ⇔ ワイヤーフレーム切替（初期状態: ワイヤーフレーム） |
| 自動回転ボタン | カメラの自動旋回 ON/OFF（初期状態: ON） |
| 砲塔旋回ボタン | 主砲・副砲の自動旋回 ON/OFF（初期状態: ON。OFF で現在角度のまま停止） |
| 砲身俯仰ボタン | 砲身の自動俯仰 ON/OFF（初期状態: ON。旋回とは独立制御） |

## 📝 メモ

- 寸法は実艦準拠（全長 263m・全幅 38.9m）を 1 unit = 1m でモデリング
- Three.js 本体のみ CDN から取得するため、初回表示にはネット接続が必要です
